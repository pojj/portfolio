Falcon Technologies

Software engineer intern at Falcon Technologies to build a database of Real-Estate licenses. Each Canadian province and American state issue slightly differing Real-Estate licenses; therefore, verifying the license of an Agent from an unknown State or Province is very time consuming. I was tasked with fixing this problem by currating an aggregated database.

  • Used Webscraping to gather licensee data from Provinces and States.
  • Parsed data using Python scripts.
  • Stored normalized data into a Dockerized PostgreSQL database.
  • Scheduled scripts to rerun ensuring up-to-date data.

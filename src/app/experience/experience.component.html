Falcon Technologies

I worked as an intern software engineer at Falcon Technologies to curate a complete database of North Amercian real estate licenses. Each Canadian province and American state manages its own license catalog making it very time-consuming to verify if an agent has a valid license. I worked to address this challenge.

  • Employed web scraping to gather licensee data from provinces and states.
  • Parsed data ingestion of all forms into normalized formats through data piplines.
  • Stored normalized data into a Dockerized PostgreSQL database.
  • Scheduled scripts to ensure the data remains up-to-date.

Delta Controls
